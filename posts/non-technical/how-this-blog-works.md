I've created this blog to experiment with how I can use AI to make blogging more interesting. Coding with AI makes the ideas behind the code more interesting than the code itself, and that makes the ideas worth storing somewhere. So I thought I would store them here.

The blog lives in a folder on GitHub, served through GitHub's free hosting. Every post is a plain text file. I write the post; an AI assistant does the supporting work — looking up the projects I mention, reading the relevant code, turning project names into links, and attaching footnotes that open a specific piece of source code right there in the reader's browser.

The assistant also lays out the text, files the post in the right place, and ships the updated blog. And it rewrites the post in a version without the programmer jargon, for people who aren't programmers. This is that version.

For the past year I've been experimenting with a style of building software where the rules live inside the tool itself, instead of being repeated in instructions to an AI every time. That keeps things from drifting between runs. I'll be writing about these tools here as they come together. They're all free and open on [my GitHub](https://github.com/niclaslindstedt).

I don't usually blog, but when I do, it's mostly a tech demo.
